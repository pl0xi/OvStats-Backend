# OvStats Website
This is a general website for OvStats, also working as a backend service for (https://github.com/pl0xi/OvStats-Twitch_Panel_League_of_Legends)

## Tech stack
- ASP.NET Core 7
- xUnit
- Moq

## Features
- REST API
  - Riot Informations
    - League summoner stats 
      - Match History
      - Rank
